BusyBee – Productivity App

UI Design • Mobile App • Prototype

Overview

BusyBee is a productivity app that helps users stay on top of tasks, manage their time, and keep focused using a bright, playful visual language.

My Role

UI design, visual direction, layout, and interaction design for the core screens.

Goals

  • Keep the interface simple and friendly for everyday use.
  • Make task management feel light instead of overwhelming.
  • Use a strong, memorable color system to support the “bee” concept.

Key Decisions

  • Used a bold yellow base with strong contrast to make tasks and actions easy to scan.
  • Rounded shapes and soft shadows to keep the experience approachable and less “corporate”.
  • Clear typography hierarchy so priorities, deadlines, and next actions stand out at a glance.

Process

Understand the problem

Identified user pain points around task management and turned them into clear needs.

Explore concepts

Explored different metaphors for productivity and selected the bee-themed direction.

Map flows & structure

Planned simple task flows for creating tasks, viewing today, and tracking progress.

Wireframe the screens

Created low-fidelity layouts to define structure and hierarchy before visuals.

Refine visual design

Finalized color palette, typography, and UI components to match the BusyBee brand.

Build the prototype

Developed an interactive Figma prototype to test key flows and micro-interactions.
