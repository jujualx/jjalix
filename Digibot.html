DigiBot – Application

UI Design • Branding • Web App

Overview

DigiBot is a concept application designed around digital creativity, collaboration, and learning. The interface highlights a bold color palette, playful illustration, and a clean layout that supports productive work without feeling too serious.

My Role

Visual direction, UI design, layout, and creating the illustration-focused hero section for the main experience.

Design Goals

  • Create a friendly, welcoming interface for creative users.
  • Use color and illustration to communicate energy and collaboration.
  • Keep the layout simple enough to support future feature growth.

Key Decisions

  • A split layout with a strong hero illustration on the right to balance typography and UI components on the left.
  • Rounded shapes, soft shadows, and warm accent colors to keep the brand approachable.
  • Clear hierarchy between headings, supporting text, and calls to action.
