Say Hello to DigiBot

UI Design • Brand Design • Web Development

Your AI-powered digital assistant, designed to engage and assist users seamlessly.

Overview

DigiBot is an AI-powered chatbot platform designed to provide instant support, answer questions, and guide users through digital experiences. The goal was to create a friendly, approachable interface that makes AI feel accessible rather than intimidating.

My Role

End-to-end product design and development — from concept, branding, and UI/UX design to front-end implementation and live deployment.

Goals

  • Create a friendly and trustworthy AI interface.
  • Design a clean, modern layout that feels easy to use.
  • Ensure the chatbot experience feels conversational and natural.

Key Decisions

  • Used soft colors and rounded components to make interactions feel welcoming.
  • Kept layouts minimal so users can focus on the conversation.
  • Designed a flexible UI that can scale for different chatbot use cases.

Process

Research & Planning

Defined the chatbot’s purpose, audience, and feature set to ensure the experience solves real user needs.

Brand & UI Design

Created the DigiBot brand identity, color system, and UI components, followed by high-fidelity screen designs.

Build & Launch

Developed the live website using HTML and CSS, tested responsiveness, and deployed the project via GitHub Pages.
